 SauceDemo Automation Testing

This project tests the*SauceDemo website using Selenium, Java, and TestNG.

 What’s Inside?
- Login Test  
- Product Selection Test 
- Cart Test
- Checkout Test 

## 🛠️ What You Need Before Running?
- Java JDK 11+
- Maven
- Google Chrome & Chromedriver

 How to Run?
mvn test
